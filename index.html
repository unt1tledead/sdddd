Задача 1
…; 12; x; 6; 3; … — найти x
Находим d
d = 3 − 6 = −3
Находим x
x = 12 + (−3) = 9
Ответ
x = 9
Задача 2
Найти a9, если a1 = 7 и d = 4
Формула n-го члена
an = a1 + (n−1)·d
Подставляем
a9 = 7 + 8·4 = 7 + 32 = 39
Ответ
a9 = 39
Задача 3
Сумма первых 16 членов АП: −8; −4; 0; …
Дано
a1 = −8,  d = 4,  n = 16
Формула
Sn = n2(2a1 + (n−1)·d)
Подставляем
S16 = 162(2·(−8) + 15·4) = 8·(−16 + 60) = 8·44 = 352
Ответ
S16 = 352
