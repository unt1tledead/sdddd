Прогрессии

Тест · Вариант I

Задача 1
…; 12; x; 6; 3; … — найти член прогрессии x
Дано: АП: …; 12; x; 6; 3; …
Найти: x
Находим d
d = 3 − 6 = −3
Формула: aₙ = a₁ + (n−1)·d
x = 12 + (−3) = 9
Ответ
x = 9
Задача 2
Найти девятый член АП, если a1 = 7 и d = 4
Дано: a1 = 7,  d = 4
Найти: a9
Формула
an = a1 + (n − 1)·d
Подставляем
a9 = 7 + (9 − 1)·4 = 7 + 32 = 39
Ответ
a9 = 39
Задача 3
Найти сумму первых 16 членов АП: −8; −4; 0; …
Дано: a1 = −8,  d = 4,  n = 16
Найти: S16
Формула
Sn = (2a1 + (n−1)·d)·n2
Подставляем
S16 = (2·(−8) + 15·4)·162 = 44·162 = 352
Ответ
S16 = 352
Задача 4
Найти первый член ГП, если b4 = 162 и q = −3

162 = b1·(−3)3 = b1·(−27)

b1 = 162−27 = −6
Ответ

Задача 6
Первый член ГП равен 11, знаменатель равен 2. Найти S5
Дано: b1 = 11,  q = 2
Найти: S5
Формула
Sn = a1·(1 − qn)1 − q
Подставляем
S5 = 11·(1 − 25)1 − 2 = 11·(−31)−1 = 341
Ответ
S5 = 341
Задача 7
Вставить 3 числа между 16 и 116, чтобы получилась ГП
Дано: b1 = 16,  b5 = 116
Найти: b2, b3, b4
Находим q
b5 = b1·q4  →  q4 = 11616 = 1256  →  q = 14
Три числа
b2 = 16·14 = 4  ;  b3 = 4·14 = 1  ;  b4 = 1·14 = 14
Ответ
16; 4; 1; 14; 116
Задача 8
Найти S5 ГП, если b5 = 81, b3 = 36, q > 0
Дано: b5 = 81,  b3 = 36,  q > 0
Найти: S5
Находим q
b5b3 = q2 = 8136 = 94  →  q = 32
Находим b₁
b3 = b1·q2  →  36 = b1·94  →  b1 = 16
Формула суммы
Sn = a1·(1 − qn)1 − q
Подставляем
S5 = 16·(1 − (32)5)1 − 32 = 16·(−21132)−12 = 211
Ответ
S5 = 211
